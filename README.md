## 二、选香港大带宽服务器，到底在看什么

市面上讲香港服务器的文章不少，但很多一上来就甩套餐表格，不告诉你怎么选。我觉得有必要先把几个核心维度讲清楚，不然后面看表格也是一头雾水。

**1. 共享带宽 vs 独享带宽**

这是最容易搞混的一点。共享带宽标的是"峰值"，比如"1G峰值"，意思是最高能跑到1G，但实际上是多人共享一条大带宽，高峰期可能跑不满。优点是便宜，适合流量可预测、不追求时刻满速的场景。

独享带宽标的是"独享"，比如"100M独享"，意思是这100M就是你一个人的，任何时刻都保证跑满。价格贵不少，但适合对带宽稳定性要求高的生产环境。

**2. 入口线路 vs 出口线路**

香港服务器的延迟，很大程度上取决于入口线路（大陆到香港这段）的质量。出口线路（香港到海外）决定了国际访问体验。好的入口线路能让大陆访问延迟压到1~2ms，差的线路可能绕一大圈，延迟飙到几十甚至上百ms。

**3. 流量计费 vs 带宽计费**

流量计费就是给你一个流量额度（比如2TB/月），带宽是峰值限制，超量要么停机要么扣费。带宽计费就是独享固定带宽，流量不限。前者适合流量波动大、偶尔爆发的业务，后者适合持续高吞吐的业务。

**4. 端内延迟**

这是很多人忽略的指标。端内延迟指的是入口节点到出口节点之间的网络延迟，直接决定了大陆用户访问的响应速度。广港专线能做到1~2ms，沪日专线大概25~28ms，沪美专线120ms以上。延迟越低，体验越"丝滑"。
